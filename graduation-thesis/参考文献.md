## 格式说明

### 文献类型标识
- [M] - 专著（图书）
- [J] - 期刊文章
- [C] - 会议论文
- [D] - 学位论文
- [R] - 研究报告
- [S] - 标准
- [P] - 专利
- [EB/OL] - 网络资源

### 引用格式模板

**期刊论文**：
```
[序号] 作者. 文章标题[J]. 期刊名称, 年份, 卷(期): 起止页码.
```

**会议论文**：
```
[序号] 作者. 文章标题[C]//会议论文集名称. 出版地: 出版者, 年份: 起止页码.
```

**专著**：
```
[序号] 作者. 书名[M]. 版本. 出版地: 出版社, 年份.
```

**网络资源**：
```
[序号] 作者/机构. 标题[EB/OL]. (发布日期)[引用日期]. URL.
```

**技术标准**：
```
[序号] 标准制定机构. 标准名称[S/OL]. 发布日期. URL.
```

## 使用建议

1. **按引用顺序编号**：在论文中按照引用的先后顺序给参考文献编号

2. **引用标注**：在正文中引用时使用上标，如：WebSocket协议[2]提供了...

3. **多处引用**：同一文献在多处引用时，使用相同的编号

4. **作者数量**：
   - 1-3位作者：全部列出
   - 4位及以上：列出前3位，后加"等"或"et al."

5. **中英文混排**：
   - 中文文献在前，英文文献在后
   - 或按引用顺序统一编号

6. **URL访问日期**：网络资源需注明最后访问日期

7. **DOI标识**：学术论文建议添加DOI（如有）

## 论文中的引用示例

### 示例1：单个引用
```
WebSocket协议[2]是HTML5标准的一部分，为Web应用提供了全双工通信能力。
```

### 示例2：多个引用
```
分布式系统的设计需要考虑CAP定理[13,35]，在一致性、可用性和分区容错性之间做出权衡。
```

### 示例3：范围引用
```
近年来，国内外学者对即时通讯系统进行了大量研究[11-18]。
```

### 示例4：作者年份引用（可选）
```
Zhang等人[13]提出了一种基于消息队列的可靠性保障机制。
```

## 注意事项

1. **真实性**：确保所有引用的文献真实存在且可查证
2. **时效性**：优先引用近3-5年的文献
3. **权威性**：优先引用核心期刊、顶级会议、知名出版社的文献
4. **相关性**：引用的文献应与研究内容直接相关
5. **完整性**：参考文献信息要完整准确
6. **格式统一**：全文参考文献格式保持一致
